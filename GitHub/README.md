# Preconditions
* Install github3 library

    `pip3 install github3.py`

* Update Configuration Area in the code files
```
############# Configuration Area #########################
# Put your GitHub token here
githubToken = ""
# GitHub ID that you want to copy from
copyFromId = ""
# GitHub ID that you want to copy to
copyToId = ""
# GitHub Organization
orgName = ""
##########################################################
```

# How to run
`python3 filename.py`
